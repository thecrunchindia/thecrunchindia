# Project Colors Guide

Here are the custom colors configured in our project using **Tailwind CSS v4** (`src/index.css`):

## 🟡 Primary
**Hex:** `#f9a602`
Our main branding color used for primary buttons, active states, and highlights.

* **Background:** `bg-primary`
* **Text:** `text-primary`
* **Border:** `border-primary`

---

## ⚫ Dark
**Hex:** `#0A0A0A`
Our primary dark shade used for deep backgrounds, heavy text, and contrasting elements.

* **Background:** `bg-dark`
* **Text:** `text-dark`
* **Border:** `border-dark`
